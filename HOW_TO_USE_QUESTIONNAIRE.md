# How to Use the Client Questionnaire

## Option 1: Fillable HTML Form (Easiest!)

1. **Open the file**: Double-click `CLIENT_QUESTIONNAIRE.html` to open it in your web browser
2. **Fill it out**: Click on any field and type your answers
3. **Save/Print**:
   - **To PDF**: Click "Print to PDF" button, or press `Ctrl+P` (Windows) / `Cmd+P` (Mac) and choose "Save as PDF"
   - **To Print**: Click "Print to PDF" button and select your printer
   - **Save Data**: Click "Save as HTML" to save a copy with all your answers

## Option 2: Convert to Fillable PDF

### Using Online Tools (Free):
1. Go to one of these websites:
   - **PDF24** (https://tools.pdf24.org/en/html-to-pdf)
   - **ILovePDF** (https://www.ilovepdf.com/html-to-pdf)
   - **SmallPDF** (https://smallpdf.com/html-to-pdf)

2. Upload the `CLIENT_QUESTIONNAIRE.html` file
3. Convert to PDF
4. Use Adobe Acrobat or an online tool to make it fillable:
   - **PDFEscape** (https://www.pdfescape.com) - Free online PDF editor
   - **Sejda** (https://www.sejda.com/html-to-pdf) - Free PDF tools

### Using Adobe Acrobat (If you have it):
1. Open the HTML file in a browser
2. Print to PDF
3. Open the PDF in Adobe Acrobat
4. Go to Tools → Prepare Form
5. Acrobat will automatically detect form fields

## Option 3: Use Google Forms

1. Go to Google Forms (https://forms.google.com)
2. Create a new form
3. Copy questions from `CLIENT_QUESTIONNAIRE.md`
4. Share the form link with your client
5. Responses will be collected automatically in a Google Sheet

## Option 4: Microsoft Word

1. Open `CLIENT_QUESTIONNAIRE.md` in a text editor
2. Copy the content
3. Paste into Microsoft Word
4. Add form fields:
   - Go to Developer tab → Insert → Text Form Field
   - Or just use regular text boxes (simpler)
5. Save as Word document or PDF

## Recommended Approach

**For easiest client experience:**
1. Send them the `CLIENT_QUESTIONNAIRE.html` file
2. Ask them to open it in their browser
3. They can fill it out directly
4. They can print to PDF or take screenshots
5. They send it back to you

**For professional look:**
1. Convert HTML to PDF using online tool
2. Make it fillable using PDFEscape or similar
3. Send fillable PDF to client
4. They fill it out and send back

## Tips

- The HTML form works best in Chrome, Firefox, or Edge browsers
- All fields are editable - just click and type
- The form will print nicely to PDF with all formatting preserved
- Required fields are marked with a red asterisk (*)

## Need Help?

If your client has trouble with any of these methods, you can:
- Fill it out together over a video call
- Use the markdown version (`CLIENT_QUESTIONNAIRE.md`) and fill it out in a simple text editor
- Create a Google Form version (takes 10-15 minutes to set up)
